# 🫀 PulseLink – Real-Time Organ Matching & Emergency Transport Coordination System

**PulseLink** is a cloud-native web platform that facilitates real-time organ donation matching and emergency transport coordination. Built using **Next.js** (frontend) and **Ballerina** (backend), this system aims to bridge the communication gap between donor hospitals, transplant centers, and emergency services—helping save lives through fast, data-driven decisions.

---

## ⚙️ Tech Stack

- **Frontend:** Next.js (App Router), TypeScript, Tailwind CSS
- **Backend:** Ballerina (developed separately)
- **API Communication:** REST (planned)
- **Additional Tools:** React Hook Form, Zustand/Redux (optional), Google Maps API, SMS Gateway (Twilio or equivalent)
